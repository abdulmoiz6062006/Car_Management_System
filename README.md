# Car_Management_System
A C++ &amp; Qt-based desktop system for vehicle inventory and sales management. Features Admin, Staff, and Customer roles with real-time stock updates and financial reporting. Built using strict OOP principles (Encapsulation, Composition) with file persistence to ensure sales and data are saved automatically.
